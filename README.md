# 自定义Snippets
## Markdown
| 快捷键 | 创建的文字                                   |
|--------|----------------------------------------------|
| `,d`   | 行内代码块                                   |
| `,c`   | 行间代码快                                   |
| `,n`   | ---                                          |
| `,b`   | **粗体**                                         |
| `,s`   | ~~被划去~~ 的文字                            |
| `,m`   | -[] 清单                                     |
| `,p`   | 图片                                         |
| `,a`   | [链接]()                                     |
| `,1`   | # H1                                         |
| `,2`   | ## H2                                        |
| `,3`   | ### H3                                       |
| `,4`   | #### H4                                      |
| `,f`   | 去往下一下`<++>` （占位符）                  |
| `,w`   | 去往下一个`<++>` （占位符）并帮你按下`Enter` |
